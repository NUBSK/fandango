Keyboard shortcuts:

play / pause: spacebar,
stop: s or escape key,
forward: f or arrow right,
rewind: r or arrow left,
volume up: + or arrow up,
volume down: - or arrow down,
mute: m,
next: n,
previous: b

API features with their options:

audioStr: /path/to/files, // Server path to audio files or URL to media server
autoPlay: false, // Boolean ; Accessibility recommendation is autoplay off
microdata: 0, // 0 = false ; 1 = audio book schema ; 2 = music album and playlist schema
pageHead: true, // Boolean ; Include metadata in head of document
transcription: 0, // 0 = false ; 1 = VTT parsing for novels ; 2 = VTT parsing for songs
helpBar: true, // Boolean ; Display bar with Settings and Help buttons
slider: false, // Boolean ; Display progress bar above player buttons
statusBar: true, // Boolean ; Display status bar
useFlash: false, // Boolean ; Get priority over HTML 5
webAudio: false // Boolean ; This feature is experimental. USE AT YOUR OWN RISK
